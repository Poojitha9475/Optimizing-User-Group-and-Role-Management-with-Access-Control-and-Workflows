# ServiceNow RBAC & Workflow Automation Project

## Overview
This project demonstrates implementation of Role-Based Access Control (RBAC) and Workflow Automation in ServiceNow Developer Instance.

## Platform
- ServiceNow Developer Instance
- Flow Designer
- Access Control (ACL)
- Role-Based Access Control (RBAC)

## Features Implemented

### User Management
- Alice (Project Manager)
- Bob (Team Member)

### Group Management
- Project Management Group
- Development Team Group

### Role Configuration
- Project Manager Role
- Team Member Role

### Access Control
- Table-level ACL
- Create, Read, Write, Delete restrictions
- Role-based permission enforcement

### Workflow Automation
Task Lifecycle:
1. Task Creation
2. Assignment
3. In Progress
4. Submit for Review
5. Approval / Rejection
6. Closure

## Security Implementation
- Principle of Least Privilege
- Role-based authorization
- Controlled task modification
- Approval-based workflow enforcement

## Project Outcome
- Improved accountability
- Structured permission management
- Secure task lifecycle automation
